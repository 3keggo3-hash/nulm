# Support and feedback

Nulm is an early public alpha. We want to hear what works and what does not.

## Report a bug

Use the [bug report template](https://github.com/3keggo3-hash/nulm/issues/new?template=bug_report.md).

## Request a feature

Use the [feature request template](https://github.com/3keggo3-hash/nulm/issues/new?template=feature_request.md).

## Questions and ideas

Prefer [GitHub Discussions](https://github.com/3keggo3-hash/nulm/discussions) for setup help and open-ended feedback.

If Discussions are not enabled yet, open a general issue or email the maintainer listed on [PyPI](https://pypi.org/project/nulm/).

## Before you open an issue

- Nulm version: `nulm --version`
- Python version: `python3 --version`
- MCP client and OS
- Redact API keys and local paths when pasting config
