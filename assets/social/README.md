# Nulm social assets (X / Twitter)

## X upload sizes

| Asset | File | Size |
|-------|------|------|
| Profile photo | `nulm-avatar.png` or `nulm-avatar.svg` | 400×400 (displays as circle) |
| Header | `nulm-header.png` or `nulm-header.svg` | 1500×500 recommended |
| Post image | `nulm-post-launch.png` | 16:9 or 1200×675 |

PNG files are generated for quick upload. SVG files are editable source (Inkscape, Figma import).

## Profile setup checklist

1. **Name:** `Nulm` or `Nulm · local MCP`
2. **Handle:** `@nulm` or `@nulmdev` (see `docs/social-launch.md`)
3. **Bio:**
   ```
   Local MCP server — guarded file/shell access + audit logging.
   Claude Desktop · Cursor · VS Code
   pip install nulm
   ```
4. **Link:** `https://github.com/3keggo3-hash/nulm`
5. **Avatar:** `nulm-avatar.png`
6. **Header:** `nulm-header.png`
7. **Pinned post:** use copy in `docs/social/x-posts.md` + attach `nulm-post-launch.png`
